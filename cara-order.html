Cara Order

Simple, cepat, dan dibantu admin sampai selesai.

  1. Pilih produk (Sprei / Bedcover / Paket / Sarung)
  2. Pilih ukuran sesuai kebutuhan
  3. Klik tombol WhatsApp pada produk
  4. Admin akan mengirim pilihan motif yang ready
  5. Admin kirim total harga + info pengiriman
  6. Pesanan diproses & dikirim

Tips

Supaya order lebih cepat diproses.

Pastikan ukuran Misal: 160x200 atau 180x200.
Pilih bahan Embos / Mikrotek / Katun Lokal.
Tanya motif ready Admin akan kirim opsi motif terbaru.
